# this-is-the-China-website

将国际网站伪装成中国网站.

这样你就可以在(半)公共区域使用国际网站, 而不会被萌新在微信 / QQ 上问, "哇! 为什么你可以使用 Google / YouTube / ...? 浇浇我!"
